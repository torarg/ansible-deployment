ansible_deployment.cli_helpers.update_deployment

ansible_deployment.cli_helpers.update_deployment(deployment, scope, non_interactive, sources_override)

Helper function for the cli update command.

Parameters

  • deployment (Deployment) – Deployment object.

  • scope (str) – Update scope.

  • scope – Scope of update. May be: all roles playbook inventory group_vars ansible_cfg

  • non_interactive (bool) – Whether or not to update will be interactive.

  • sources_override (sequence) – Override inventory sources.
